Help

This page can help you to know what each page is doing

Select what do you want

Add New Student

This page can help you to add a new student to the system, you should follow specific criteria to add new student data, like: student's name should be added without any numbers, student's id should be 8 numbers without any letters, student's GPA should be from 0 to 4, student should be greater than 16 years old, phone number should have 11 number (e.g., 0110 123 4567), student's level must be fourth or third if he have specific department (else you can select None) and you should select student's gender and status.

Edit Student Data

This page can help you to update or delete an existing student information with the same criteria that used in “Add Student” page but you cannot update student department, if you want to change department you should use "Search" and "Student's Department Assignment" page.

Students Information

This page view all active/inactive students in your system, you can also change student' status from this page.

Search

This page can help you find any active student by his name, after search you can change student's department through "Student's Department Assignment" page.

Student's Department Assignment

This page can help you to assign department to student after searching, this page has student's id, name and list of departments but the student should be "Third Level" or "Fourth Level" to do select any department from this page, you can reach this page only after searching.
